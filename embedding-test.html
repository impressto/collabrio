About This Test

This page has intentionally aggressive CSS that tries to override many common HTML elements:

  • All headings are forced to be red and uppercase
  • All buttons are forced to be red with specific styling
  • All inputs/textareas have blue borders and gray backgrounds
  • All divs have red dashed borders

The Collabrio app below should maintain its own styling despite these overrides.
